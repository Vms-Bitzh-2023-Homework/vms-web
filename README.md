# vms-web

## 包管理器
- pnpm

## 构建工具
- Vite

## 技术栈
- TypeScript
- Vue3

## 组件
- axios
- vue-router
- element-plus

## 下载 & 配置
```sh
npm install pnpm -g
git clone https://github.com/Vms-Bitzh-2023-Homework/vms-web.git
cd vms-web
pnpm install
```

## Dev
```sh
pnpm run dev
```

## Build
```sh
pnpm run build
```
